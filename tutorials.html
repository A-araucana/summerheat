Leaflet Tutorials

Preparation for the Summer Heat Web Application

These are the final products of tutorials completed in preparation for building the Summer Heat Web Application. Each tutorial focuses on specific aspects of the Leaflet mapping library.

  • 1. Quickstart Tutorial

    Fundamentals of Leaflet mapping.

    Learning Outcomes:

    • Map initialization and configuration
    • Adding tile layers from OpenStreetMap
    • Creating markers, circles, and polygons
    • Binding interactive popups to features
    • Handling map click events
    View Tutorial →

  • 2. GeoJSON Tutorial

    Working with geographic data formats.

    Learning Outcomes:

    • Understanding GeoJSON structure (points, lines, polygons)
    • Creating custom styled markers and shapes
    • Applying data-driven styling based on properties
    • Filtering features conditionally
    • Working with feature attributes and metadata
    View Tutorial →

  • 3. Adapted Tutorial

    Loading external data sources.

    Learning Outcomes:

    • Using fetch() API to load external GeoJSON files
    • Working with asynchronous JavaScript and promises
    • Organizing code with functions (createMap, getData)
    • Generating dynamic popups from feature properties
    • Handling DOM events for proper initialization
    View Tutorial →
