GeoJSON Tutorial

What this tutorial demonstrates:

This tutorial shows how to work with GeoJSON data in Leaflet maps, including different geometry types and styling options.

Point Features: Baseball stadiums (Coors Field, Busch Field) as point markers
LineString Features: Two colored line features showing how to draw paths
Polygon Features: State boundaries colored by political party (Republican=red, Democrat=blue)
Custom Styling: Different colors, weights, and opacities for various features
Interactive Popups: Click on features to see popup content with details
Conditional Rendering: Features filtered by properties (show_on_map)
Custom Markers: Circle markers with custom styling instead of default pins
Data-Driven Styling: Colors based on feature properties (political affiliation)

Note: This tutorial requires the 'map' variable to be defined. You may need to add basic map initialization code.
